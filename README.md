## Usage
`git clone`, then `mvn install` inside this repo's root. Use as Maven dependency in your projects:
```xml
<dependency>
  <groupId>com.babai.dui</groupId>
  <artifactId>dui</artifactId>
  <version>1.0-SNAPSHOT</version>
</dependency>
```
